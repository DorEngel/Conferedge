This folder holds self-hosted copies of the company logos shown on the page.

To populate it, run from the publish/ folder:
    bash fetch-logos.sh

Expected files (filenames must match exactly):
    zimark.svg, descope.svg, doit.png, imper.png, intel.svg,
    linx-security.webp, doorloop.svg, solaredge.png, teva.png

You can also drop in your own/official versions manually using these names.
